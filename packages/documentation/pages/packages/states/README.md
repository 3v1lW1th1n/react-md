## @react-md/states

Create different user interaaction states for elements such as:

- keyboard only focus
- hover
- pressed
- selected

### Installation

```sh
$ npm install --save @react-md/states
```

It is also recommended to install the following packages:

```sh
$ npm install --save @react-md/theme \
    @react-md/typography \
    @react-md/utils
```
