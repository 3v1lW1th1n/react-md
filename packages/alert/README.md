# @react-md/alert

Create accessible alerts.

## Installation

```sh
$ npm install --save @react-md/alert
```

It is generally recommended to also install the following packages since they
work hand-in-hand with this package:

```sh
$ npm install --save @react-md/theme \
    @react-md/typography \
    @react-md/button
```
